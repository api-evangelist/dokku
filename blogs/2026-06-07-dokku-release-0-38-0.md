Dokku 0.38.0 introduces significant architectural improvements including a migration of the builds plugin to Go with per-build record tracking, consolidation of environment file paths, and a storage plugin redesign treating volumes as named first-class resources. The release addresses multiple security vulnerabilities in patch 0.38.2 including command injection prevention and hardened archive extraction against symlink traversal. New features include buildpack specification via app.json, live-restore enabled by default on fresh installs, and per-process Docker options for enhanced resource management.
